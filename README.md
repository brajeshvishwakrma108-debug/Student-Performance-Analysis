# Student Performance Analysis

This project analyzes student performance data using Python, SQL, Excel, and Power BI.

## Tools Used
- Python (Pandas, NumPy)
- MySQL
- Power BI
- Excel

## Project Workflow
1. Load Excel data in Python
2. Data cleaning and preprocessing
3. Store cleaned data in SQL database
4. Connect SQL with Power BI
5. Build interactive dashboard

## Key Insights
- Test preparation improves student performance
- Parental education impacts scores
- Lunch type affects academic results
